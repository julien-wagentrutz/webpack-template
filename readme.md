#Webpack template

##Features
- Basic Files Support (HTML,CSS, Stylus,JS,Images,Fonts)
- Extracted CSS
- Transpilled  JS (With BAbel)
- Sourcemaps
- Clean Dist
- Dev server (with local network support)
- Static Files
- Caches breaker 

## Instructions 
Install depencies

```npm i```

Launch dev server

```npm run dev```

Build for production

```npm run build```
